# DemoQA Home Project

Проект для автоматизации тестирования с использованием Selenium и Python.

## Структура проекта

- `pages/` - Page Object классы
- `tests/` - тестовые сценарии  
- `components/` - компоненты страниц
- `conftest.py` - фикстуры pytest
